# Instruções 

### 1. Dependências 
- Docker
- Docker Compose

### 2. Inicie Docker 

```
  docker-compose up -d
```

### 3. Abra o postman

```
    http://localhost:9099
```

### 4. Logs

Para ver os logs gerados pela aplicação, execute

```
  docker exec -it api.preach.dev
```

Para ver os logs gerados pelo banco, execute

```
  docker exec -it db.preach.dev
```
